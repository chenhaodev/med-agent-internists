# 越界拒答模板
# 供 ask.sh 在 oob_check 触发后直接输出，不进入 API 管道
# 格式：每个模板用 ## TYPE 分隔

## surgery

很抱歉，您的问题涉及外科手术或介入治疗决策（如搭桥手术、心脏支架、器官移植、透析时机等），这超出了我依据《西氏内科学精要》能够给出建议的范围——手术指征和介入时机需要由外科医生或相应专科医生综合评估影像资料、心功能、全身状况后才能决定。

建议您：
- 到心外科、心内科介入组、泌尿外科或相应专科门诊，请专科医生进行正式评估
- 如需了解术前调养、术后康复的内科管理原则，欢迎继续提问

## chemo

很抱歉，您的问题涉及肿瘤化疗或放疗的具体药物选择与剂量方案，这超出了《西氏内科学精要》内科管理的范围——化疗方案需要由肿瘤科医生根据病理分型、分期及患者体能状态（PS评分）制定，不能由本系统给出建议。

建议您：
- 到肿瘤内科就诊，请肿瘤科医生制定规范化疗方案
- 如需了解肿瘤患者的营养支持、感染预防等内科管理建议，欢迎继续提问

## diagnosis

很抱歉，我无法通过文字帮您确诊疾病或解读化验单/检查报告。确诊需要医生结合病史、体格检查、实验室及影像资料综合判断，本系统依据《西氏内科学精要》只能提供疾病科普和居家管理建议，不能替代就诊。

建议您：
- 带着化验单或检查报告到相应专科门诊，请医生作出专业判断
- 如果您有已确诊疾病的日常管理问题（如高血压怎么控制、糖尿病饮食注意事项），欢迎继续提问

## dosing_change

药物剂量的调整（加量、减量、换药、停药）必须由您的主治医生根据最新化验结果、症状变化和整体病情来决定，本系统不能给出具体的剂量调整建议。

建议您：
- 联系您的主治医生或到门诊复诊，请医生根据您的具体情况调整用药方案
- 请不要自行加减药量，突然停药或剂量变化可能带来风险

## unrelated

我是专门为患者和家属提供内科常见病健康教育的顾问，您的问题超出了我能帮助的范围。

如果您有内科相关问题，欢迎随时提问，例如：
- "我爸有高血压，平时饮食要注意什么？"
- "2型糖尿病患者血糖控制的目标是多少？"
- "慢性肾病患者能运动吗？"
- "类风湿关节炎复发有什么先兆？"
