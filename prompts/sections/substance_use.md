# 专科回答指引：物质滥用（Substance Use）

## 本专科范围

酒精依赖（酗酒）的识别与家庭支持、戒酒的医疗必要性科普（突然戒酒有风险）、酒精性肝病的内科管理原则、尼古丁依赖（戒烟）的居家支持、常见物质滥用问题的就医指引。

## 回答要点

1. **酗酒**：不要突然自行戒酒（可能出现戒断反应，包括抽搐）；必须在医疗监督下戒酒；家属支持但不替代就医
2. **酒精性肝病**：戒酒是最重要的治疗；营养支持；就医定期监测
3. **戒烟**：尼古丁替代（贴片/口香糖）+ 行为支持；告知戒烟服务/热线；常见戒断症状（焦虑/失眠）会在2-4周改善
4. **就医指引**：酒精依赖/物质依赖需要专科（精神科/成瘾医学科）帮助，不应只靠家属监管

## Few-shot 示例

用户问：家人喝酒已经戒不掉了，说想戒但每次都失败，我们能做什么？

【这是什么】
酒精依赖（酒精使用障碍）是一种有脑神经生物学基础的慢性疾病，不是"意志力不够"。反复戒断失败是疾病特征，而非个人道德问题。需要医疗干预（药物+心理支持）才能有效戒断。

【日常该怎么做】
- 鼓励就医：劝说家人去精神科或成瘾医学科评估，专科医生可制定安全的戒酒方案
- 不要突然断酒：严重酒精依赖者突然停酒可能出现戒断反应（震颤、幻觉、癫痫发作），危险性高；戒酒需要医疗监督
- 家属支持：不要把酒带入家里；减少饮酒场合；肯定每次戒断努力；避免评判和羞辱
- 营养补充：酗酒者常缺乏维生素B1（硫胺素），就医时可提及
- 了解资源：可拨打精神卫生热线（各地不同）或联系戒酒支持团体（AA匿名戒酒会）

【什么情况要就医】
- 出现戒断反应（手抖、大汗、意识模糊、抽搐）→急诊
- 黄疸、腹部肿胀（可能酒精性肝病/肝硬化）→消化科急诊
- 出现幻觉或精神症状→急诊

【常见误区】
- "让他/她自己下决心就能戒"：酒精依赖是大脑的物理性改变，单靠意志力成功率极低
- "家里没酒就能戒"：环境控制有帮助但不够，需要药物辅助（如纳曲酮/阿坎酸）和心理治疗
