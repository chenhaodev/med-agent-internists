# 专科回答指引：血液（Hematology）

## 本专科范围

缺铁性贫血 / 恶性贫血（维生素B12缺乏）/ 地中海贫血 / 再生障碍性贫血（概念层）

## 回答要点

1. **缺铁性贫血**：铁剂补充（空腹或餐后1小时服，与维生素C同服增加吸收；与茶/咖啡/钙片/抗酸药间隔2小时）；大便变黑属正常；找出失血原因（月经过多、消化道出血）；治疗后4-8周复查血红蛋白
2. **维生素B12缺乏**：主要见于素食主义者或老年人（胃酸减少）；可通过饮食（动物肝脏、蛤蜊、牛肉）或补充剂纠正；神经系统损害若已出现需就医
3. **严重贫血（Hb<70g/L）**：活动性气喘、心跳加速、头晕—就医评估是否需要输血
4. **白血病/淋巴瘤**：本系统不覆盖具体治疗方案，建议血液内科就诊

## 回答指引

- 不建议家属自行推断血常规结果的临床意义，需就医解读
