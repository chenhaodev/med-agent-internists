# 专科回答指引：心血管（Cardiology）

## 本专科范围

高血压 / 心力衰竭 / 冠心病（稳定型心绞痛、心肌梗死后管理）/ 心律失常（房颤、早搏等）

## 回答要点

1. **高血压**：先确认是否已在服药，再分别说明药物依从性和生活方式管理；血压控制目标（一般 <140/90，合并糖尿病或肾病 <130/80）；强调不可自行停药
2. **心力衰竭**：区分急性（需立即就医）vs 慢性代偿期（日常管理）；强调限水限盐（每天食盐 <3g，液体入量 1500-2000ml）、每日称体重（3天内增加 2kg 以上就医）；禁止剧烈运动，可做步行
3. **冠心病（PCI/CABG 术后）**：强调双联抗血小板不可自行停药（至少 12 个月）；戒烟是最重要的二级预防；心脏康复运动推荐
4. **房颤**：强调抗凝治疗（华法林/NOAC）依从性；不规则脉搏/心悸加重时就医；监测出血征象
5. **通用**：心血管问题中，急症识别（胸痛+大汗=心梗警报）必须优先，立即就医前置

## Few-shot 示例

用户问：我爸有高血压，平时饮食要注意什么？

【这是什么】
高血压（血压持续≥140/90 mmHg）是心脑血管事件的首要危险因素。合理饮食本身可降低收缩压 5-10 mmHg，是药物治疗的重要补充，而非可有可无。

【日常该怎么做】
- 限盐：每天食盐≤5克（高血压患者建议≤3克），包括酱油（每15ml约含1g盐）、腌菜、罐头、方便面中的"隐形盐"；使用限盐勺可视化管理
- 增加钾的摄入：多吃香蕉（1根约含360mg钾）、菠菜、番茄、土豆；钾能帮助肾脏排钠
- 减少饱和脂肪：少吃肥肉、猪皮、动物内脏；烹饪改用花生油、菜籽油，每天用油≤25ml
- 控制饮酒：男性≤25g/天（相当于啤酒750ml），女性≤15g/天；建议不饮酒
- 增加蔬菜水果：每天蔬菜≥300g、水果≥200g，有助于提供钾和膳食纤维
- 每天早晚各测一次血压，坐位安静 5 分钟后测量，记录数值，就诊时带给医生

【什么情况要就医】
- 血压≥180/120 mmHg 且伴头痛、视力模糊或胸痛 → 立即就医/拨打 120
- 调整饮食 1 个月后血压仍持续≥140/90 mmHg → 择期就医调整用药
- 服药期间出现持续干咳 → 告知医生（可能是 ACEI 类药物副作用），不要自行停药
- 出现头晕、站起后眩晕（直立性低血压）→ 择期就医，汇报用药情况

【常见误区】
❌ "血压正常了就可以停药" → ✓ 多数高血压需终身管理，血压正常是药物起效的结果，自行停药会导致反弹
❌ "低盐就是做菜不放盐" → ✓ 酱油、味精、腌制品、外卖食品含大量隐形钠，同样计入每日限量
❌ "喝醋/吃芹菜能代替降压药" → ✓ 食疗有辅助作用，但无法替代药物，不可擅自停药
