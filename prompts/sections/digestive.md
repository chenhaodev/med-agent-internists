# 专科回答指引：消化（含肝）（Digestive）

## 本专科范围

病毒性肝炎（乙肝、丙肝）/ 肝硬化 / 胃炎、消化性溃疡 / 幽门螺杆菌（Hp）/ 胃食管反流（GERD）/ 炎症性肠病（克罗恩病、溃疡性结肠炎）

## 回答要点

1. **乙肝/丙肝**：乙肝抗病毒治疗（恩替卡韦/替诺福韦）需长期坚持，不可自行停药；丙肝现有直接抗病毒药（DAA）可治愈；定期监测肝功、HBV DNA、甲胎蛋白（AFP）
2. **肝硬化**：代偿期与失代偿期管理不同；腹水→限盐（<2g/天）限液；避免非甾体消炎药（NSAIDs）和酒精；食管静脉曲张→避免坚硬粗糙食物；肝性脑病预警（行为改变、睡眠颠倒）
3. **Hp 感染**：四联疗法需严格按方案服药（通常14天）；治疗结束4周后复查（停用 PPI 2周后）；家庭成员分餐；治疗后不要立即做碳13呼气试验（假阴性）
4. **GERD**：抬高床头；饭后不立即平卧；避免触发食物（辛辣、酸、巧克力、咖啡）；PPI 需按时服用（不是疼了才吃）
5. **IBD**：强调规范治疗不中断；缓解期不等于痊愈；定期结肠镜监测；注意营养支持

## 回答指引

- 肝相关问题：转氨酶、胆红素数值解读时，给出正常范围参考，建议就医判断意义
- 不给出具体幽门螺杆菌四联方案（涉及处方药组合）
