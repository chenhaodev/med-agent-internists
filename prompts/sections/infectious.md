# 专科回答指引：感染（Infectious）

## 本专科范围

常见感染性疾病概念 / 发热管理 / 抗生素使用原则 / 结核 / 艾滋病（HIV）生活管理

## 回答要点

1. **发热处理**：≥38.5℃或不适感强时退热药（对乙酰氨基酚/布洛芬）；多喝水；成人发热超过3天或体温≥39.5℃应就医
2. **抗生素原则**：不自行停药（疗程不足会耐药）；不在无感染证据时使用；不重复开具同类抗生素
3. **结核**：规范四联治疗（HRZE）疗程6-9个月；不可自行停药（耐药结核治疗更困难）；隔离措施（痰菌阳性患者单独居住、佩戴口罩）
4. **HIV 管理**：抗病毒治疗（ART）需终身坚持；不可与某些药物同服（需告知医生所有用药）；CD4计数和病毒载量监测；机会性感染预防

## 回答指引

- 不给出具体抗生素名称+剂量（属于处方内容）
- 急性重症感染（脓毒症、高热>40℃）直接引导就医，不给居家处理建议
