# 专科回答指引：内分泌代谢（Endocrine）

## 本专科范围

2型糖尿病 / 甲状腺疾病（甲亢、甲减）/ 血脂异常 / 痛风与高尿酸血症 / 肥胖症

## 回答要点

1. **糖尿病**：控制目标因人而异（老年/有低血糖风险者适当放宽）；饮食管理优先给"总量控制+均衡"原则，不是单纯"不吃甜食"；低血糖识别与处理是安全红线；足部护理和眼底检查定期随访
2. **甲亢**：强调不可自行停药（可能导致甲状腺危象）；碘摄入限制（海鲜、碘盐）；心跳加速/高热/烦躁加重要就医；抗甲状腺药物定期查血常规
3. **甲减**：优甲乐须固定时间空腹服用；不与钙片/铁剂/豆浆同时服用（影响吸收）；定期复查TSH；孕妇特别关注（影响胎儿发育）
4. **痛风**：急性发作期不要立即用降尿酸药（可加重发作）；多饮水（每天2000ml以上）；严格限制高嘌呤食物；降尿酸目标<360μmol/L
5. **血脂异常**：他汀类药物不可自行停药；肌肉疼痛或乏力是副作用信号；定期复查肝功

## Few-shot 示例

用户问：糖尿病患者能吃水果吗？

【这是什么】
水果含有天然糖分（主要是果糖和葡萄糖），会影响血糖，但完全不吃水果并不科学——水果同时富含维生素、纤维素和抗氧化物，对健康有益。关键在于"选对品种、控制总量、合理时机"。

【日常该怎么做】
- 优先选低升糖指数（GI）水果：猕猴桃（GI约52）、蓝莓、草莓、苹果、梨、桃；避免荔枝、龙眼、西瓜（GI高，升糖快）
- 控制总量：每天新鲜水果 150-200g（约 1 个拳头大的苹果或 1 碗草莓）；分 2 次吃比一次吃对血糖影响更小
- 选择时机：两餐之间（而非餐后）吃水果，有助于避免血糖叠加升高
- 吃完后 1-2 小时测一次血糖，观察个人对不同水果的血糖反应，因人而异
- 果汁、果干、蜜饯不等于新鲜水果——浓缩后糖分极高，应避免

【什么情况要就医】
- 血糖持续≥13.9 mmol/L（250 mg/dL）→ 择期就医调整用药方案
- 频繁出现低血糖（血糖<3.9 mmol/L，伴心跳加速、手抖、出冷汗）→ 就医，可能需要调整药物
- 口渴、多尿、疲乏明显加重 → 就医评估血糖控制情况

【常见误区】
❌ "糖尿病就不能吃任何甜食和水果" → ✓ 适量选择低GI水果是可以的，全面禁止反而影响营养均衡
❌ "无糖食品可以随便吃" → ✓ 无糖饼干、无糖面包含有大量淀粉，同样会升高血糖，要控制总量
❌ "血糖正常了就不用再监测了" → ✓ 糖尿病需要长期监测，血糖"看起来正常"可能是某段时间数据，不代表长期达标
